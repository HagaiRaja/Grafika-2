----------------------------------------
EXTRACT IMAGE
----------------------------------------
on linux terminal type
python jpgToTxt.py

make sure you have the picture with 
filename "image.jpg" is in the same
folder with the terminal
----------------------------------------





----------------------------------------
COMPILE
----------------------------------------
on linux terminal type
gcc image.c -o <program name>

example
gcc image.c -o g
----------------------------------------





----------------------------------------
RUN
----------------------------------------
on linux interface press
Ctrl + Alt + F1

go to file directory
./<program name>

example 
./g
----------------------------------------
